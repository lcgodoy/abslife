# Internal helper for residual present value

Internal helper for residual present value

## Usage

``` r
residual_pv(residual_value, lifetime, elapsed_time, discount_factors)
```

## Arguments

- residual_value:

  Contractual residual value.

- lifetime:

  Time of termination.

- elapsed_time:

  Current age.

- discount_factors:

  Vector of discount factors.
