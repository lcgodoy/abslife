# Variance of the log-transformed hazard estimate

Variance of the log-transformed hazard estimate

## Usage

``` r
var_hat(lambda, risk_set, fh, n)
```

## Arguments

- lambda:

  hazard rate

- risk_set:

  \\hat{U}\\

- fh:

  \\hat{f}\\

- n:

  sample size (or number of timepoints)

## Value

a scalar
