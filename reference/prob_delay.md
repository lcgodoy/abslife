# Internal helper for probability of delay

Internal helper for probability of delay

## Usage

``` r
prob_delay(delay, lifetime, elapsed_time, max_delay, delay_prob_matrix)
```

## Arguments

- delay:

  Length of delay.

- lifetime:

  Time of termination.

- elapsed_time:

  Current age.

- max_delay:

  Maximum possible delay.

- delay_prob_matrix:

  Delay probability matrix.
