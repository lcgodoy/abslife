# Carry Forward Last Observation for Zero-Hazard Estimates (Internal Use)

Carry Forward Last Observation for Zero-Hazard Estimates (Internal Use)

## Usage

``` r
fix_0haz(haz_est)
```

## Arguments

- haz_est:

  A `data.frame` calculated using the function `estimate_hazard`

## Value

A `data.frame` with hazard estimates carried forward when zeros are
present.
