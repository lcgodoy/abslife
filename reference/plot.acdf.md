# Plot Method for an 'acdf' Object

Plot Method for an 'acdf' Object

## Usage

``` r
# S3 method for class 'acdf'
plot(x, ...)
```

## Arguments

- x:

  An object of class `acdf`. Typically the output of the
  `estimate_hazard` function.

- ...:

  Additional arguments passed to the base `plot` function (e.g., `xlab`,
  `ylab`, `ylim`).

## Value

A plot of the CDF.
