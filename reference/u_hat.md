## Usage

``` r
u_hat(t, time_to_event, trunc_time)
```

## Arguments

- t:

  A time point at which hazard estimates are sought.

- time_to_event:

  A numeric vector representing the observed time to event.

- trunc_time:

  A numeric vector representing the observed left-truncated time.

a scalar
