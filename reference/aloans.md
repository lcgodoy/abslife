## Format

### `aloans`

A data frame with 58,118 rows and 7 columns:

- `risk_cat`:

  Credit score category

- `Z`:

  Time-of-event

- `Y`:

  Truncation time

- `C`:

  Censor indicator (1 = censored, 0 = not censored)

- `D`:

  Default indicator (1 = default, 0 = not default)

- `bond`:

  Categorical variable specifying different bonds.
